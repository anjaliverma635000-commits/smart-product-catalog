# Smart Product Catalog

A Spring Boot project demonstrating core backend concepts.

## Features
- Spring Boot
- Spring Data JPA
- H2 Database
- Spring Security
- JWT Authentication
- AOP Logging
- Dynamic Proxy
- Caching
- REST APIs

## Tech Stack
- Java 17
- Spring Boot 3
- Gradle
- H2 Database

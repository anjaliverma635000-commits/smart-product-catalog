# Smart Product Catalog

A Spring Boot REST API project that demonstrates modern backend development using Spring Boot, Spring Security, JWT Authentication, Docker, and GitHub Actions.

---

## 🚀 Features

* Spring Boot REST API
* Spring Security
* JWT Authentication
* BCrypt Password Encryption
* DTO Validation
* Global Exception Handling
* Pagination & Sorting
* Spring AOP Logging
* Spring Cache
* Swagger (OpenAPI)
* JUnit 5 & Mockito Testing
* Docker
* GitHub Actions (CI/CD)

---

## 🛠️ Tech Stack

* Java 17
* Spring Boot 3
* Spring Security
* Spring Data JPA
* H2 Database
* JWT
* Gradle
* Docker
* Swagger (OpenAPI)
* JUnit 5
* Mockito
* GitHub Actions

---

## ▶️ Run the Project

```bash
./gradlew bootRun
```

---

## 🧪 Run Tests

```bash
./gradlew test
```

---

## 🐳 Docker

Build Docker Image:

```bash
docker build -t smart-product-catalog .
```

Run Docker Container:

```bash
docker run -p 8080:8080 smart-product-catalog
```
